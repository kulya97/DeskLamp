# 简介

自己随手diy的一个星球灯。离计划目标还有一段距离。

主要是控制一个8串的ws2812，代码比较简单。可以切换不同的颜色和亮度，同时可以作为呼吸灯使用，夜里用非常的炫。

主控使用的时stc8f2k，元件用的全都是手头有的，比较浪费，可以自己替换。

3D打印的模型使用下面的滚花螺母连接。
